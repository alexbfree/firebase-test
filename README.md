# firebase-test
Just Testing Firebase Realtime Database

Deployed at [https://test-firebase-db-ajbb.web.app/]

using command
```
/usr/local/Cellar/node/7.0.0/bin/firebase deploy --only hosting:test-firebase-db-ajbb
```
